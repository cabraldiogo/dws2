About me

  • I am a Web Design and Development student at Full Sail University. As you can see at the menu bar my passions are: Web, Music and Sports. I already took 6 months of classes, and these are the Web Courses that I took so far.

  • Web Programming Fundamentals

    The Web Programming Fundamentals course introduces students to basic programming and logic skills.

  • Programming for Web Applications I

    The Programming for Web Applications I course trains students in the technologies used to create dynamic content for the web using client-side programming.

  • Designing Computer Graphics

    The Designing Computer Graphics Course is designed to teach students basic levels of graphics creation

  • Designing for Web Standards I

    Designing for Web Standards I examines the process of creating functional, standards-based content for the Internet.

  • Advanced Computer Graphics

    Advanced Computer Graphics is designed to teach students advanced levels of graphics creation through the use of software programs that are used by design, animation, and interactive media companies worldwide.

  • Web Interface and Usability

    The Web Interface and Usability Course teaches students the concepts and strategies needed to create successful web interfaces
